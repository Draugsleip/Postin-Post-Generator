# PostIn Post Generator

AI powered LinkedIn post generator that transforms URLs into a post or creates a unique post based on real time web trends also provides voiceover for the posts created.

# Prerequisites

- `OpenAI API`

- `Google Cloud API`

- `Google Search Engine ID`

- Register an application on the **[LinkedIn Developer Platform](https://developer.linkedin.com)** to obtain the following credentials:

  - `LinkedIn Client ID`

  - `LinkedIn Client Secret`

  - `Linkedin Access Token` (Expires every 60 days)

# What's New in v2

- **Direct Share:** It is now possible to share a post directly with a single click rather than the copy-paste approach used in the main branch.

- **Session Logout:** You can logout by going to localhost:5000/logout to invalidate the previous token belonging to session.
